# Directory Overview

This directory contains a collection of Markdown files, which are notes and tutorials related to Linux command-line tools and infrastructure as code. The main topics covered are:

*   **Zsh and Oh My Zsh:** Setting up the Z shell, Oh My Zsh framework, and the Powerlevel10k theme.
*   **Zsh Plugins:** A curated list of useful Zsh plugins for improving productivity.
*   **Terraform:** A beginner's tutorial on Terraform for infrastructure as code.

# Key Files

*   `Setting Up Zsh with Oh My Zsh & Powerlevel10k on Ubuntu.md`: A step-by-step guide to installing and configuring Zsh, Oh My Zsh, and the Powerlevel10k theme on Ubuntu.
*   `The Only 6 Zsh Plugins You Ever Need.md`: A list of recommended Zsh plugins with installation instructions and usage examples.
*   `Terraform_tutorial.md`: A tutorial for getting started with Terraform, from basic concepts to deploying a simple AWS EC2 instance.

# Usage

This directory serves as a personal knowledge base for command-line tools and DevOps practices. The files can be used as a reference for setting up a new development environment or for learning about the covered topics.
